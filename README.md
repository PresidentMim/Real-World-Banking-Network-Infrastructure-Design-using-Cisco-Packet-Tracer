I designed and implemented a secure, scalable, and efficient network infrastructure for Bangladesh Bank and 6 major commercial banks using Cisco Packet Tracer.

🔹 Project Highlights

✅ Designed a hierarchical network architecture connecting:

Bangladesh Bank (Core Router)

Sonali Bank

Janata Bank

Agrani Bank

Dutch-Bangla Bank

Islami Bank

Rupali Bank

✅ Implemented Variable Length Subnet Masking (VLSM) for efficient IP allocation across 18 departmental subnets.

✅ Configured VLAN Segmentation to separate:

HR Department

Finance/Operations Department

IT Department

✅ Implemented OSPF Dynamic Routing Protocol for fast convergence, scalability, and reliable communication between all bank branches.

✅ Used Router-on-a-Stick (RoAS) for inter-VLAN routing.

✅ Established secure WAN connectivity using point-to-point serial links.

✅ Successfully verified:

Intra-VLAN communication

Inter-VLAN communication

Inter-bank communication

End-to-end network connectivity through extensive ping testing

🔹 Technologies Used

Cisco Packet Tracer

OSPF

VLAN

VLSM

Router-on-a-Stick (RoAS)

Layer 2 Switching

WAN Serial Links

Network Design & Troubleshooting

This project helped me gain practical experience in enterprise-level network design, routing, subnetting, VLAN implementation, and troubleshooting techniques used in real-world banking environments.
